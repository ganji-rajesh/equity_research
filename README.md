# Fundamental Analysis Research Repository

A curated collection of company analysis materials generated by 
processing annual reports using Large Language Models (LLMs).

## Why This Repo?
Fundamental analysis — covering economic, industry, and company-level 
research — is deeply qualitative and time-consuming. It requires reading 
through hundreds of pages across multiple sources. LLMs now make it 
possible to rapidly synthesize this information without losing depth.

## What's Here?
I conduct fundamental analysis of 1–2 listed Indian companies every month.
This repo contains structured summaries and insights derived from:
- **Statutory report sections** from company annual reports
- **Financial statements** sourced from MoneyControl

## Tools Used
- [Management Discussion Summarizer](https://managementdiscussionsummary-ganjirajesh.streamlit.app/)
- Google Gemini models
